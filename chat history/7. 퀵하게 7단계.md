## 7단계 UI/UX 개선 계획

현재 상태를 보면:
- 기본적인 라우팅과 네비게이션이 구현되어 있음
- 각 상태관리 방식별 Todo 앱이 구현되어 있음
- 기본적인 스타일링은 있지만 개선이 필요함

### 개선해야 할 부분들:

1. **공통 스타일링 개선**
   - 각 상태관리 방식별로 구분되는 시각적 디자인
   - 일관된 색상 테마와 타이포그래피
   - 카드 형태의 레이아웃으로 개선

2. **각 버전별 구분을 위한 UI 개선**
   - useState, useReducer, useContext, Redux, Zustand 각각 다른 색상 테마
   - 각 방식의 특징을 나타내는 아이콘이나 배지

3. **반응형 디자인 적용**
   - 모바일, 태블릿, 데스크톱 대응
   - 그리드 레이아웃 개선

4. **사용자 경험 최적화**
   - 로딩 상태 표시
   - 애니메이션 효과
   - 더 나은 인터랙션 피드백

현재 ask 모드이므로 코드를 직접 수정할 수는 없지만, 다음과 같은 개선 방향을 제안드립니다:

### 제안하는 개선 사항:

1. **색상 테마별 구분**
   - useState: 파란색 계열
   - useReducer: 초록색 계열  
   - useContext: 보라색 계열
   - Redux: 주황색 계열
   - Zustand: 핑크색 계열

2. **컴포넌트 스타일링 개선**
   - 카드 형태의 레이아웃
   - 그림자 효과와 호버 애니메이션
   - 더 나은 버튼 디자인

3. **반응형 그리드 시스템**
   - CSS Grid와 Flexbox 활용
   - 미디어 쿼리를 통한 반응형 디자인

## 주요 개선 사항:

1. **색상 테마별 구분**: 각 상태관리 방식마다 고유한 색상 테마 적용
2. **모던한 디자인**: 글래스모피즘 효과, 그라디언트, 그림자 효과
3. **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
4. **애니메이션**: 부드러운 전환 효과와 호버 애니메이션
5. **사용자 경험**: 더 나은 시각적 피드백과 인터랙션
6. **접근성**: 포커스 스타일과 키보드 네비게이션 개선

import React, { useState, useEffect } from 'react';
import type { Todo as TodoType } from '../types/todo';
import TodoList from './TodoList';
import TodoForm from './TodoForm';
import { loadTodosFromStorage, addTodoToStorage, updateTodoInStorage, deleteTodoFromStorage } from '../utils/storage';
import { createTodo } from '../utils/helpers';

const UseStateTodoApp: React.FC = () => {
  const [todos, setTodos] = useState<TodoType[]>([]);

  // localStorage에서 todos 불러오기
  useEffect(() => {
    const savedTodos = loadTodosFromStorage();
    setTodos(savedTodos);
  }, []);

  const addTodo = (text: string) => {
    const newTodo = createTodo(text);
    addTodoToStorage(newTodo);
    setTodos(loadTodosFromStorage());
  };

  const toggleTodo = (id: string) => {
    setTodos((prev) => {
      const updatedTodos = prev.map((todo) => (todo.id === id ? { ...todo, completed: !todo.completed } : todo));
      const updatedTodo = updatedTodos.find((todo) => todo.id === id);
      if (updatedTodo) {
        updateTodoInStorage(updatedTodo);
      }
      return loadTodosFromStorage();
    });
  };

  const deleteTodo = (id: string) => {
    deleteTodoFromStorage(id);
    setTodos(loadTodosFromStorage());
  };

  return (
    <div className="todo-app-container use-state-todo-app">
      <h2 className="todo-app-title">useState Todo App</h2>
      <TodoForm onAdd={addTodo} />
      <TodoList todos={todos} onToggle={toggleTodo} onDelete={deleteTodo} />
    </div>
  );
};

export default UseStateTodoApp;
